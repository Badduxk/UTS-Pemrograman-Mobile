**OMNI** merupakan aplikasi berbasis **Flutter** yang dikembangkan untuk memenuhi tugas **Ujian Tengah Semester (UTS)** pada mata kuliah **Pemrograman Mobile**.

Aplikasi ini terdiri dari beberapa halaman utama, yaitu:
